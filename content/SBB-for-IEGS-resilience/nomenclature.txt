Case Format
# ----------------------------------------------------------------------- #
1. Electricity power system parameters
- ebus: electricity bus
    0          id  bus id (no.)
    1           x  x-coordinate (scaled)
    2           y  y-coordinate (scaled)
    3      p_load  real power load demand (MW)
    4      q_load  reactive power load demand (MVAr)
    5     v_level  voltage level (kV)
    6       v_min  lower bound of bus voltage magnitude (p.u.)
    7       v_max  upper bound of bus voltage magnitude (p.u.)
    8          gs  shunt conductance demanded at V = 1.0 p.u. (MW)
    9          bs  shunt susceptance injected at V = 1.0 p.u. (MVAr)
    10    penalty  penalty cost of load shedding ($/MWh) 
- line: branch
    0          id  line id (no.)
    1        head  from bus number (no.)
    2        tail  to bus number (no.)
    3           r  resistance (p.u.)
    4           x  reactance (p.u.)
    5           b  total line charging susceptance (p.u.)
    6         tap  transformer off nominal turns ratio, if non-zero (p.u.)
    7       rateA  long term rating A, set to 0 for unlimited (MW)
# ----------------------------------------------------------------------- #
2. Gas transmission system parameters
- node: node
    0          id  node id (no.)
    1           x  x-coordinate (scaled)
    2           y  y-coordinate (scaled)
    3      f_load  gas load demand (kg/s) 
    4       p_min  minimum node pressure (Pa) 
    5       p_max  maximum node pressure (Pa) 
    6     penalty  penalty cost of load shedding ($/kg) 
- pipe: pipeline
    0          id  pipeline id (no.)
    1        head  from node number (no.)
    2        tail  to node number (no.)
    3      length  length (m)
    4    diameter  diameter (m)
    5   roughness  roughness (mm) 
    6       f_min  minimum gas flow (kg/s)
    7       f_max  maximum gas flow (kg/s)
    8      lambda  friction factor (p.u.)
- link: short pipe and resistor
    0          id  link id (no.)
    1        head  from node number (no.)
    2        tail  to node number (no.)
    3       f_min  minimum gas flow (kg/s)
    4       f_max  maximum gas flow (kg/s)
- comp: compressor
    0          id  compressor id (no.)
    1        head  from node number (no.)
    2        tail  to node number (no.)
    3       p_min  minimum node pressure rate (p.u.) 
    4       p_max  maximum node pressure rate (p.u.) 
    5       f_min  minimum gas flow (kg/s) 
    6       f_max  maximum gas flow (kg/s) 
    7  fuel_ratio  ratio of gas for driving (p.u.)
- valv: control valve
    0          id  valve id (no.)
    1        head  from node number (no.)
    2        tail  to node number (no.)
    3       p_min  minimum node pressure difference (Pa) 
    4       p_max  maximum node pressure difference (Pa) 
    5       f_min  minimum gas flow (kg/s) 
    6       f_max  maximum gas flow (kg/s) 
- well: gas well
    0          id  gas well id (no.)
    1        node  node number (no.)
    2       f_min  minimum gas flow (kg/s) 
    3       f_max  maximum gas flow (kg/s) 
    4        cost  gas production cost ($/kg) 
# ----------------------------------------------------------------------- #
3. Finite difference method parameters
- ends: interval endpoint
    0          id  interval endpoint id (no.)
    1     pipe_id  index affiliation of pipeline (no.)
    2   head_node  index affiliation of head-end node, -1 if not (no.)
    3   tail_node  index affiliation of tail-end node, -1 if not (no.)
- segs: segment
    0          id  segment id (no.)
    1        head  head-end interval end point number (no.)
    2        tail  tail-end interval end point number (no.)
    3     pipe_id  index affiliation of pipeline (no.)
    4      length  length of segment (m)
    5    diameter  diameter (m)
    6        area  area (m^2)
    7      lambda  friction factor (p.u.)
# ----------------------------------------------------------------------- #
4. Generator parameters
- gens: generator
    0          id  generator id (no.)
    1         bus  bus number (no.)
    2       p_min  minimum real power output (MW)
    3       p_max  maximum real power output (MW)
    4       q_min  minimum reactive power output (MVAr)
    5       q_max  maximum reactive power output (MVAr)
    6     ramp_10  ramp rate for 10 minute reserves (MW)
    7     ramp_60  ramp rate for 60 minute reserves (MW)
    8       cost2  generation cost c2 ($/MWh^2)
    9       cost1  generation cost c1 ($/MWh)
    10      cost0  generation cost c0 ($)
- unit: gas-fired unit
    0          id  gas-fired unit id (no.)
    1         bus  bus number (no.)
    2        node  node number (no.)
    3       p_min  minimum real power output (MW) 
    4       p_max  maximum real power output (MW) 
    5       q_min  minimum reactive power output (MVAr) 
    6       q_max  maximum reactive power output (MVAr) 
    7     ramp_10  ramp rate for 10 minute reserves (MW)
    8     ramp_60  ramp rate for 60 minute reserves (MW)
    9  efficiency  energy conversion efficiency (kg/s/MW)
# ----------------------------------------------------------------------- #
5. Auxillary parameters
- slot: load curve (given by the percentage of maximum load demand)
    0          id  slot id (no.)
    1  time_stamp  o'clock time (hour)
    2      e_load  electricity load demand (p.u.)
    3      g_load  gas load demand (p.u.)
- fail: simulated position of failures
    0          id  failure scenario id (no.)
    1        prob  scenario probability (p.u.)
    2      fail_1  line id of the 1st failed line (no.)
    3      fail_2  line id of the 2nd failed line (no.)
    4      fail_3  line id of the 3rd failed line (no.)
- base: base unit
    0          id  base unit id (no.)
    1      base_V  voltage (kV) 
    2      base_S  power (MVA) 
    3      base_Z  resistance (Ohms) 
    4      base_I  current (kA) 
    5      base_F  gas flow (kg/s) 
    6      base_P  gas pressure (Pa)
    7      base_W  lifting (kg^2/s^2/Pa)
